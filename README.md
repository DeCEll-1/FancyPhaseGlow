Customizable Phase Glow (CPG) is a small Starsector mod that lets you change and customize the "phase glow" visual effects on phase-capable ships. It adds an in-game UI in the refit screen so you can pick from multiple glow presets and tweak visual parameters per-ship - no coding or shader knowledge required for evyday use.

## Installation

For a detailed walkthrough, troubleshooting tips, and screenshots, see the full installation guide on the Fractal Softworks forum: [Fractal Softworks Forum](https://fractalsoftworks.com/forum/index.php?topic=25974.0)

How to use in-game after instalation

1. Open the refit screen for a ship that is phase-capable.
2. Open the hullmod menu
3. Select "Customizable Phase Glow" category from the bottom
4. Add the "Custom Phase Glow" hullmod
5. Click the "Modify Phase Effects" button (bottom of the refit UI).
6. In the dialog that opens:
   - Use the dropdown to select a glow preset (these come from the mod itself and from other installed mods that provide presets).
   - Once selected, a list of tweakable elements appear. Click edit to modify individual properties.
7. Changes are saved per-ship automatically - you don't need to manually save them.
